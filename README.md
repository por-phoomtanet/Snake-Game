# 🐍 TikTok Snake Live

เกมงู AI เล่นเองสำหรับไลฟ์ TikTok — ผู้ชมส่งของขวัญเพื่อ **ตั้งชื่องู / เปลี่ยนสกิน / สร้างกำแพง / เปลี่ยนธีม**
ยิ่งเล่นยิ่งเร็วและยากขึ้น เล่นต่อเนื่องได้ไม่มีสะดุด

## ▶️ เริ่มเล่น (Phase 1 — พร้อมใช้)
เปิด `index.html` ด้วยเบราว์เซอร์ได้เลย (ดับเบิลคลิก) — ไม่ต้องติดตั้งอะไร

จะมี **แผงทดสอบ** ด้านล่างให้กดส่งของขวัญ/พิมพ์แชตทดสอบ และมีโหมด "สุ่มอัตโนมัติ"
ที่ทำให้เกมมีของขวัญ/แชตวิ่งเข้ามาเองตลอดเวลา

**ปุ่มลัด:** `H` ซ่อน/แสดงแผงทดสอบ · `M` เปิด/ปิดเพลง · `N` เปิด/ปิดเอฟเฟกต์

> 🔊 เสียงจะเริ่มหลัง **คลิก/กดปุ่มครั้งแรก** (นโยบายบล็อกเสียงอัตโนมัติของเบราว์เซอร์)
> เพลงพื้นหลัง + เอฟเฟกต์สร้างจาก Web Audio API ไม่ต้องมีไฟล์เสียง

## 🎥 ใช้กับ OBS
1. OBS → Sources → **＋ → Browser**
2. เลือก **Local file** ชี้ไปที่ `index.html`
   - หรือใส่ URL: `.../index.html?obs=1` เพื่อซ่อนแผงทดสอบอัตโนมัติ
3. ตั้งขนาด **Width 960 / Height 540** (ปรับได้ที่ `js/config.js` → `grid`)
4. HUD / ลีดเดอร์บอร์ด / ทอสต์ ลอยอยู่บนจอพร้อมสตรีมทันที

พารามิเตอร์ URL:
- `?obs=1` ซ่อนแผงทดสอบ
- `?auto=1` เริ่มด้วยโหมดส่งของขวัญอัตโนมัติ (ค่าเริ่มต้นคือ **กดเอง**)

## 🔌 ต่อ TikTok จริง (เฟสถัดไป)
สถาปัตยกรรมแยกแหล่งข้อมูลผ่าน `SG.Bus` แล้ว — ตัวจำลองกับ TikTok จริงส่ง event รูปแบบเดียวกัน:
```js
SG.Bus.emit('gift', { user, giftId, qty });
SG.Bus.emit('chat', { user, message });
```
ขั้นตอน:
1. ตั้ง backend Node ด้วย [TikTok-Live-Connector](https://github.com/zerodytrash/TikTok-Live-Connector) (ไลบรารีไม่เป็นทางการ)
2. ส่ง event มาที่เบราว์เซอร์ผ่าน WebSocket
3. **map giftId ของ TikTok → คีย์ใน `SG.GIFTS`** แล้ว emit เข้า `SG.Bus`
4. ปิดตัวจำลอง: เปิดด้วย `?auto=0` และไม่เรียก `SG.Simulator`

ดูโครงเริ่มต้นได้ที่ `server/tiktok-server.js`

> ⚠️ TikTok-Live-Connector เป็น API ไม่เป็นทางการ อาจใช้งานไม่ได้ตามอัปเดตของ TikTok
> และ **ห้ามใส่ ANTHROPIC_API_KEY ในโค้ดฝั่งเบราว์เซอร์** — เก็บไว้ใน backend เท่านั้น
